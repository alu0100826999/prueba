Pandoc

Pandoc

Introducción

Pandoc es una herramienta de codigo abierto que nos facilita la conversión de documentos de forma rapida y efectiva.

Instalación.

Para su instalación en mac os, será necesario descargar el ejecutable proporcionado desde la página web oficial de Pandoc, y seguir las instrucciones proporcionadas por este.

Uso Basico.

Algunos comandos básicos para el uso de Pandoc son:

  • pandoc -f markdown -t html -s -o . Nos permite convertir un documento del formato markdown al formato html.
  • pandoc -f html -t markdown -s -o . Nos permite convertir un documento del formato html al formato markdown.
  • pandoc --help. Proporciona algunos comandos básicos de pandoc, así como sus posibles opciones.
